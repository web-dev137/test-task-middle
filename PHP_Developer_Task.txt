Задача: Разработать сервис конвертации валют.

Сервис должен уметь:
1. Собирать и парсить данные минимум из одного указанного источника;
2. Иметь GUI для конвертации суммы из одной валюты в другую;
3. GUI должно быть доступно только после прохождения процедуры аутентификации пользователя (логин-пароль);
4. Учесть, что количество запросов на конвертацию может быть большим, или оригинальный источник курсов может быть недоступен, по этой причине не стоит на каждый запрос дергать сайт ЦБ РФ.

Источники курсов:
1. ЦБ Тайланда: ЦБ Тайланда: https://apiportal.bot.or.th/bot/public/node/503, API KEY: c2bbe063-d0ff-456c-bc08-fbd5115fb340, 200 запросов в день.
2. ЦБ России: http://www.cbr.ru/scripts/XML_daily.asp

Дополнительно (выполнять не обязательно, но будет плюсом):
- Получение курсов выполнить в виде cli команды, запускаемой по крону;
- Добавить в GUI построение исторических графиков по курсам валют. К примеру на highcharts отображать график движения курса выбранной пары на текущую неделю.


Выполненное задание необходимо разместить в публичном репозитории, снабдив README.md файлом, с описанием процедуры установки/запуска.
